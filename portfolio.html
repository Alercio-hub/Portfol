Case Study: Building an Indemnity Calculator for IPAJ

Project Overview

I developed an interactive Indemnity Calculator for the Instituto de Patroc√≠nio e Assist√™ncia Jur√≠dica (IPAJ) using HTML, CSS, and JavaScript. The goal was to create a simple yet effective web-based tool to help legal professionals, employees, and employers calculate indemnities based on Mozambican labor laws.

The Challenge

Many people face difficulties in determining their rightful compensation due to:

  • Complex labor laws requiring manual calculations.
  • Frequent errors in computing severance payments.
  • Time-consuming processes that require legal assistance for basic calculations.

I wanted to create a fully client-side solution that could generate accurate indemnity results quickly and efficiently without requiring an internet connection or backend processing.

The Approach

1. Designing the User Interface

Since simplicity and clarity were key, I built a clean, intuitive interface using:

  • HTML for structuring the input fields and results section.
  • CSS for a professional and responsive layout.
  • JavaScript for dynamic interactions and real-time calculations.

The form included:

  • Dropdowns for contract types and termination reasons.
  • Input fields for salary, years of service, and additional compensations.
  • A results section displaying the breakdown of the indemnity.

2. Implementing the Calculation Logic in JavaScript

I coded the indemnity formulas directly in JavaScript, handling different termination types:

  • Employer's Initiative (With/Without Just Cause)
  • Employee Resignation
  • Mutual Agreement

The script automatically:

  • Validated user inputs.
  • Applied the correct legal formulas.
  • Displayed a detailed breakdown of the indemnity calculation.

3. Enhancing User Experience

To make the tool more engaging, I added:

  • Real-time updates‚ÄîResults changed dynamically as users input their data.
  • Error handling‚ÄîAlerts for missing or incorrect inputs.
  • Mobile responsiveness‚ÄîEnsured usability on both desktops and smartphones.

Results & Impact

  • Fast and Accurate Calculations: Reduced reliance on manual computations.
  • User-Friendly: Even individuals without legal knowledge could use it.
  • Accessible: Since it's pure HTML, CSS, and JavaScript, it works offline without a backend.

Key Takeaways

  • JavaScript is powerful enough for legal calculations without needing a backend.
  • A clean UI improves accessibility, making legal tools more user-friendly.
  • Automation of legal processes can greatly enhance efficiency in institutions like IPAJ.

Next Steps

  • Adding a PDF export feature for users to save their results.
  • Integrating dark mode for better user experience.
  • Expanding it to include other legal calculations like vacation pay and tax deductions.

This project proved that even with only HTML, CSS, and JavaScript, I could develop a practical and impactful tool for legal professionals and the general public.
